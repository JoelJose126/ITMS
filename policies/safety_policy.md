# Abuse, Safety & Vulgarity Policy

This policy protects the system and human agents from toxic interactions (Feature 4).

## 1. Zero Tolerance for Abuse
*   **First Warning:** If an agent detects vulgarity or personal attacks, it must immediately issue a formal warning: *"I am here to help, but I must ask you to maintain a professional tone so we can resolve this together."*
*   **Hard Escalate:** On the second instance of abuse, the agent must immediately close the automated interaction and escalate the ticket to a **Security Supervisor** with a [TOXIC_USER] tag.

## 2. Legal & Fraud Threats
*   If a user mentions "Legal action", "Lawyer", or "Lawsuit", the agent must stop all autonomous processing and escalate the ticket to the **Legal Compliance Human Queue**.

## 3. Fraud Prevention
*   If an agent detects suspicious patterns (e.g., 5 refund attempts on different Order IDs in 10 minutes), it must flag the account for **Manual Fraud Review** and pause all autonomous actions.
