# Refund Policy

This policy governs all refund requests processed by the ITMS agents.

## 1. Primary Refund Windows
Eligibility is determined based on the **Reason for Return** and the time elapsed since the **Purchase Date**.

| Reason for Refund | Autonomous Window | Human Audit Required | Rejection Window |
| :--- | :--- | :--- | :--- |
| **Tampered / Damaged Package** | 1 - 7 Days | 8 - 30 Days | > 30 Days |
| **Color Mismatch / Not as Expected** | 1 - 2 Days | 3 - 14 Days | > 14 Days |
| **Change of Mind / No Longer Needed** | None | 1 - 7 Days | > 7 Days |
| **Incorrect Item Received** | 1 - 7 Days | 8 - 21 Days | > 21 Days |

## 2. Evidence Requirements
*   **Autonomous Approval:** No photo evidence required if within the autonomous window.
*   **Human Audit:** Supporting documentation (photos of the item, shipping label, or receipt) is **MANDATORY**. The agent must gather these before escalating.

## 3. Duplicate Prevention
*   Only **ONE** active refund request is permitted per Order ID. 
*   If a request is already pending or a refund has been processed, the agent must reject the new request immediately.

## 4. Processing Time (ETA)
*   Once approved, refunds are processed within **5-7 business days**.
*   The agent must inform the user of this ETA during the success or escalation message.
